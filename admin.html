URL Guide

Since this runs on GitHub Pages (static hosting), videos need to be accessible via URL:

Option A — Relative Path
assets/my-video.mp4

Upload the file to your repo's assets/ folder. Best for small files (<100MB).

Option B — External CDN
https://cdn.example.com/video.mp4

Host on Cloudinary, Bunny.net, Vimeo, or any CDN. Recommended for large/HD files. Must allow CORS.

Option C — Git LFS
assets/video-lfs.mp4

Enable Git Large File Storage in your repo for files up to 2GB on GitHub.
